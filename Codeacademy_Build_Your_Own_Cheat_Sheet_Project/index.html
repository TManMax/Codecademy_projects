HTML Tables
Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table row container
<tfoot> Table Foot The set of rows defining the footer in a table

Table Attributes
Attribute Name Description
<colspan> Column Span Defines how many columns a td element should span
<rowspan> Row Span Defines how many rows a td element should span

CSS selectors and specificity
Tag Name Description
p {} p element selector Selects the p HTML element based on the name
#p {} id="p" Uses the id attribute of the p HTML element to select a specific element
.p {} class="p" Uses the class attribute of the p HTML element to select elements which have the p class
p,h1,h2,h3 {} Paragraph, Heading 1, Heading 2, Heading 3 Groups the elements and selects all the the HTML elements with the smae style definitions
* {} * Selects all elements
Common CSS properties
Tag Name Description
border:; Border A shorthand property for border-width, border-style and border-color
color:; Color Sets the color of text
background-color:; Background-color Specifies the background color of an element
box-sizing:; Box-sizing Defines how the width and height of an element are calculated: Should they include padding and borders, or not
display:; Display Specifies how a certain HTML element should be displayed
margin-bottom:; Margin-bottom Sets the bottom of an element
margin-left:; Margin-left Sets the left margin of an element
margin-right:; Margin-right Sets the right margin of an element
margin-top:; Margin-top Sets the top margin of an element
@media:; media types Sets the style rules for different media types/devices/sizes
width:; width Sets the width of an element
height:; Height Sets the height of an element
font-size:; Font-size Specifies the font size of text
font-style:; Font-style Specifies the font style for text
font-weight:; Font-weight Specifies the weight of a font
font-family:; Font-family Specifies the font family for text
